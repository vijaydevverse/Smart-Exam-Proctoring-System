# 🎓 Smart Exam Proctoring System

An AI-powered Smart Exam Proctoring System developed using Python, OpenCV, MediaPipe, YOLOv8, and Streamlit.

## Features

✅ Face Detection

✅ Multiple Face Detection

✅ Mobile Phone Detection

✅ Automatic Screenshot Capture

✅ Violation Logging

✅ Real-Time Monitoring

✅ Streamlit Dashboard

## Technologies Used

* Python
* OpenCV
* MediaPipe
* YOLOv8
* Streamlit
* Pandas

## Project Structure

```
Smart Exam Proctoring System
│
├── face_detection.py
├── phone_detection.py
├── logger.py

│
├── screenshots/
├── logs/
└── models/
```

## How to Run

### Install Dependencies

```bash
pip install opencv-python mediapipe ultralytics pandas streamlit
```

### Run Face Detection

```bash
python face_detection.py
```

### Run Phone Detection

```bash
python phone_detection.py
```

### Run Dashboard

```bash
streamlit run dashboard.py
```
